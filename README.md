# JavaScript Project

## Aplikacja generująca przepisy z artykułów spożywczych, podanych przez użytkownika

#### Instalacja Zależności

```shell
npm install
```

#### Uruchomienie serwera deweloperskigo

```shell
nmp start
```

#### Budowanie w trybie produkcyjnym

```shell
npm run production
```
